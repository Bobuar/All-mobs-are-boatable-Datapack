# All-mobs-are-boatable-Datapack
Some mobs can not be put into a boat, because their hitbox is wider than the boat's. This datapack makes it so these mobs can be put into boats, without changing their hitboxes.

## Download
To download the pack use Modrinth - https://modrinth.com/datapack/all-mobs-are-boatable
Here will only contain the latest version of the pack on the last version of Minecraft I've updated it to, Modrinth makes it easy to select the version of Minecraft you’re using.
